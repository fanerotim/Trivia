## My 'WHY' of the project

This is a small Trivia / Quiz game, which I built after completing a TypeScript course with the main goal being practicing TypeScript. 

#### Technical Stack:
* Front-end: React JS
* API: https://opentdb.com/

The app is deployed and can be accessed via: https://fanerotim-brainy.netlify.app/

#### Features
* The main purpose of the app was to practice TypeScript, so my entire focus was in that direction
* Used Zod library to valid user input
* Built a custom Select component
* Implemented a custom hook that works with the custom Select
* Implemented Dark / Light theme
* Implemented a custom queryBuilder
* Implemented a custom shuffle algorithm (not Fisher-Yates)

#### Possible future features
* I am considering connecting a database to the app, so I can store results of users that want to play with their username. This will allow me to keep score statistics and display the top players.
